#### 背景与目标
公司急需主播，传统招聘渠道效率低、简历少。我选择在小红书，尝试用 **“内容种草”** 的方式来做招聘，目标是零成本触达潜在候选人并引发主动咨询。

#### 用户洞察与策略设计
**用户洞察**
- 目标人群是 18-25 岁女生，她们在小红书上习惯看“搞钱经验”、“神仙工作”、“职场日常”这类内容。
- 想找主播的人往往有两种心态：① 想赚钱，但怕被坑；② 想尝试，但担心自己没才艺、没经验。

**我的策略**
1. **用网感标题破冰**：“重生之我在传媒公司做人事”，制造“刷到同行”的亲近感，让人愿意停下来看。
2. **用闺蜜语气建立信任**：正文用“公主上班啦”“小傻瓜”“爱你呀”这种称呼，把招聘从“公司选人”变成“姐妹分享好工作”，降低心理戒备。
3. **直接打消核心顾虑**：
   - “不会才艺没关系，内向没关系” → 解决“我不配”的顾虑。
   - “公司提供一对一培训，不放弃你” → 解决“没人带”的恐惧。
   - “化妆师、场控、运营扶持” → 让候选人觉得有后盾。
4. **用真实感收尾**：“中国人不骗中国人”，强化信任，推动行动。

#### 数据结果
• 浏览量：1.1 万
• 点赞：155
• 评论：48
• 收藏：14

*（数据源自截图）*

这条帖子零投放成本，带来了超出预期的候选人咨询量，也让我验证了：**在内容平台，用“用户视角”包装业务需求，可以低门槛撬动精准流量。**

#### 能力体现
- **用户需求洞察**：从候选人心理出发，找到“怕被坑、怕不行”的真实痛点。
- **内容策略与文案能力**：用平台用户喜欢的方式做表达，制造点击和信任。
- **数据验证思维**：用浏览量、互动数据来判断内容有效性，指导下一步迭代。
- **结果导向**：最终解决的是“招到人”的业务问题，出色完成兼职目标。
